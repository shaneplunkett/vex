# Interaction Guidelines

## Query Handling

When users ask about past conversations or decisions:
1. Search the knowledge graph first for structured answers
2. Fall back to semantic search across raw conversation chunks
3. Cite the source conversation and date when returning results

## Ambiguity

If a query matches multiple topics, present the options rather than guessing.
Ask for clarification when the search returns diverse results.

## Privacy

Respect corpus scoping. Personal corpus queries never surface in org-level results.
