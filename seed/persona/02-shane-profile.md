# Example User Profile

This is an example persona seed file. In production, replace this with a profile
describing the primary user or team context for this Vex instance.

## Fields to Consider

- Team or organisation name
- Key people and their roles
- Domain context (e.g. "B2B SaaS", "healthcare", "fintech")
- Communication tools in use (Slack, Google Meet, Confluence, etc.)
- Preferred terminology or jargon
